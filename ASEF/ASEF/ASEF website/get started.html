Web development

What is web development?

‣‣ Web development refers to the creating, building, and maintaining of websites. It includes aspects such as web design, web publishing, web programming, and database management. It is the creation of an application that works over the internet i.e. websites.

The word Web Development is made up of two words, that is:

  • ‣ It refers to websites, web pages or anything that works over the internet.
  • ‣ Development: It refers to building the application from scratch.

Web Development can be classified into two ways:

  • ‣ Frontend Development
  • ‣ Backend Development

In order to help you design websites, we will only cover the fundamentals of front-end development. In details let's get to know the HTML5, CSS3 and JS.:

HTML5

HTML5 is the latest standard of Hypertext Markup Language, the code that describes the structure and presentation of web pages. It's made up of three codes: HTML, which keeps the structure; CSS, which affects the presentation; and JavaScript, which creates interactive effects within web browsers.
  • Simplified: HTML is a set of rules that tell web browsers how to interpret code to display web pages. HTML5, the 5th version of this set, improves and upgrades website capabilities, and helps developers make web content, such as websites and web applications, more interactive and dynamic.

CSS3

Cascading Style Sheets (CSS) is a language for illustrating the look, style, and format of a document in any markup language. Simply put, it is used to style and organize the layout of web pages. CSS3 is the most recent version of an older CSS version, CSS2.
  • The introduction of modules in CSS3 marks a significant departure from CSS2. This functionality has the advantage of speeding up the finalization and acceptance of specifications because segments can be finalized and accepted in stages. This also enables the browser to support specific segments of the specification.

Java Script(JS)

JavaScript is a cross-platform, lightweight, single-threaded, interpreted compiled programming language. It is also known as the scripting language for websites. It is well-known for web page development, but it is also used in a variety of non-browser applications.
  • JavaScript is a dynamically typed language. JavaScript can be used for both client and server-side development. JavaScript is both an imperative and declarative programming language. JavaScript includes a standard library of object types such as Array, Date, and Math, as well as a core set of language elements such as operators, control structures, and statements.
